# web-tuto-with-gin
web tutorial for practicing iot service in nclab curriculum

---

#### Create a new module
```shell
$ go mod init
```

#### Install dependencies
```shell
$ go get -u github.com/gin-gonic/gin
$ go get -u github.com/go-sql-driver/mysql
$ go get -u github.com/jinzhu/gorm
```

#### Run
```shell
$ go run main.go
```
