# web-tuto-with-gin
web tutorial for practicing iot service in nclab curriculum

---

#### Install Golang

<https://golang.org/doc/install>

#### Install Gin Web Framework

```shell
$ go get -u github.com/gin-gonic/gin
```

#### Run
```shell
$ go mod init
$ go get -u github.com/gin-gonic/gin
$ go run main.go
```
